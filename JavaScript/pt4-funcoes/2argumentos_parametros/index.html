Argumentos e Parâmetros

Imagine que uma função é como uma receita de bolo.

Parâmetros: são os espaços na receita onde você escreve o que precisa colocar.

Argumentos: são os ingredientes que você realmente coloca quando faz o bolo.

Por exemplo, na receita você ode escrever: "coloque fruta e água". Aqui fruta e água são parâmetros. Quando você faz o suco de morango você coloca morangos e 200 ml de água. Esses valores são argumentos.

Em JavaScript isso fica assim:

function fazerSuco(fruta, agua){
      console.log("Fazendo suco de " + fruta + "com" + agua + "ml de agua");
    }

    fazerSuco("morango", 200);

Aqui, fruta e agua são parâmetros.

Resumindo:

  • Parâmetros são os nomes que a função usa para receber valores.
  • Argumentos são os valores que você dá para a função quando chama ela.
