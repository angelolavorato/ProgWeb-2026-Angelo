Variáveis: let

Declaração de variáveis com let

exemplo:

let nome = "João";

Observação:

Declarar uma variável com valor usando o "let" não pode ter o mmesmo nome de uma variável já declarada anteriormente, o que ajuda a manter a consistêcia e evitar erros de programação acidentais. Por exemplo, se tentarmos declarar outra variável com o mesmo nome, o JavaScript lançará um erro de sintaxe.
