Conversão e Coerção de tipos de dados

Conversão de tipos é quando fazemos a conversao de um tipo de de dado para outro tipo. Feito de forma consciente, ou seja, quando o programador deseja a conversão, ou de forma implícita.

Coerção de tipos de dados é quando a linguagem de programação converte automaticamente um tipo de dado para outro tipo, sem que o programador tenha solicitado essa conversão. A coerção pode ocorrer em operções que envolvem diferentes tipis de dados, como comparação e concatenação.
