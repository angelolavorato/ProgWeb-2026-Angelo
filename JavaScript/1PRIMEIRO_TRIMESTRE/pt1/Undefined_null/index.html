Variáveis undefined e null

Variáveis undefined e null são valores especiais em JavaScript ou mesmo dados primitivos que representam a ausência de valor ou falta de finição.

Undefined é o valor atribuído a uma variável que foi declarada mas não inicializada. Como pode ser visto no exemplo no arquivo scripts.js.

Null é um valor atribuído a uma variável para indicar que ela não tem valor ou que o valor é desconhecido. Ele é usado para representar a ausência de um objeto ou valor. No exemplo do arquivo scripts.js, a variável nulo é atribuída com o valor null.
