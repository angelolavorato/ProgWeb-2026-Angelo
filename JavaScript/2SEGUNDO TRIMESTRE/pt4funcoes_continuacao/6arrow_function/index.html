Arrow Function

Arrow Functions são uma forma mais concisa de escrever funções em JavaScript. Elas foram introduzidas no ES6 e são especialemte úteis para funções anônimas.

A sintaxe básica de uma arrow function é:

(param1, param2) => {
      //corpo da função
    }

Se a função tiver apenas um parâmetro, os parênteses podem ser omitidos: 

(param1, param2) => param1 + param2;

Arrow Functions também têm comportamento diferente em relação ao contexto de this, o que as torna úteis em certos casos, como em callbacks. Elas não têm seu próprio this, mas herdam o valor de this do contexto onde foram definidas
