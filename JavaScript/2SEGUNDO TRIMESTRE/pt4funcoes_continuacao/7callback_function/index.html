Callback Function

Uma callback function é uma função que é passada como argumento para outra função e é executada após a conclusão de uma operação assíncrona.

As callback functions são usadas para lidar com operações que levam tempo, como requisições de rede, leitura de arquivos ou temporizadores.

Por exemplo, imagine que você queira buscar dados de um servidor. Você pode usar uma função de callback para processar os dados assim que eles forem recebidos, sem bloquear o restante do código.
